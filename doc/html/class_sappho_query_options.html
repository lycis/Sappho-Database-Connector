Detailed Description

This class is used to set any statement options like WHERE clauses or ORDER BY etc.

This class represents any kind of options that aplly to a statement. This may range from WHERE clauses to ORDER BY columns or subselects and joins.

Warning:
Currently only simple where clauses are implemented!
